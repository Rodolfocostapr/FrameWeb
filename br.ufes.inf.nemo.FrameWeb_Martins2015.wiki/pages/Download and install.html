Download and install To setup the environment for development, follow the steps below:
  1. Download Eclipse EMF according your platform;
  2. Install EMF Eclipse according this guide and run it;
  3. Open "Help >> Install Modeling Components" menu and install the necessary modeling components:
    • Acceleo;
    • Ecore Tools;
    • OBEO Designer;
    • OCL Tools;
    • Sirius.
  4. Install the Eclipse UML Plug-in. For more information about Eclipse UML Plug-in access the Eclipse MTD wiki;
  5. To work with graphical environment, open "Help >> Eclipse Marketplace" menu and install the Obeo Designer Community.
Now you are ready to import FrameWeb projects. Importing the FrameWeb Project To import FrameWeb projects to your environment you must follow the next steps:
  1. If you want to collaborete with us you can use the GitHub resourses for it;
  2. If you just want to know research and run our code you just download the necessary resources at this GitHub;
  3. Then you need to import FrameWeb EMF projects on your environment. This tutorial shows how to do it.
More information can be found at Eclipse EMF Documentation page or on EMF Developer Guide. An other good EMF tutorial can be found here.
